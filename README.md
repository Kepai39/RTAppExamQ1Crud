# RTAppExamQ1Crud

## dependancies
1. Python 
1. python 3 venv venv for virtual environment to install flask
2. Flask  (`pip install Flask`)
3. pip 
4. gunicorn (`pip install gunicorn`)
 
## running locally
1.    pip install -r requirements.txt
2.    python app.py
3.    The API will be running at http://127.0.0.1:8000
